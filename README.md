# Projeto CRUD (Python + Flask + SQLAlchemy)



### 1. Configurar o Ambiente

No terminal, crie e ative o ambiente virtual:

```bash
# Criar o ambiente virtual
py -m venv venv

# Ativar o ambiente (Windows)
.\venv\Scripts\activate
```

### 2. Instalar Dependências

Com o ambiente ativo, instale as bibliotecas necessárias:

```bash
py -m pip install Flask Flask-SQLAlchemy
```

### 3. Rodar o Projeto

Execute o arquivo principal do Flask:

```bash
py app.py
```

### 4. Acessar a Aplicação

Abra seu navegador e acesse:
[http://127.0.0.1:5000/](http://127.0.0.1:5000/)
